Introducción matemática a MASTERMIND

Completed

Este ensayo comprende una breve introducción matemática al juego de mesa Mastermind creado en 1970 por Mordecai Meirowitz. Este juego comprende adivinar un código secreto recibiendo retroalimentación ternaria a cada turno, y se juega a dos roles (aunque técnicamente la mayor diversión solo se lo lleva uno!). El ojetivo primordial del escrito es encontrar la mejor primera jugada para el jugador que adivina el código. Para lograrlo, se hace uso de estadística bayesiana, combinatorias, y un segundo análisis con enfoque frecuentista.
